## Sobre a Solução

O Project Genesis: Lunar Ark Cloud é uma plataforma hospedada no Microsoft Azure para monitoramento da Arca Lunar Inteligente.

A solução permite acompanhar indicadores operacionais da estação lunar responsável pela preservação de sementes e grãos para futuras gerações.

---

## Problema Resolvido

Mudanças climáticas, desastres naturais e crises globais podem comprometer a segurança alimentar da humanidade.

A Arca Lunar Inteligente atua como um repositório seguro de sementes e grãos fora da Terra.
